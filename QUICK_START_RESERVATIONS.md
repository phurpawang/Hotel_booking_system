# 🚀 Quick Start - Hotel Reservation Dashboard

## ⚡ Start Using in 3 Steps

### Step 1: Access the Dashboard
Login with your hotel account and navigate to:

- **Owner**: Go to `/owner/reservations`
- **Manager**: Go to `/manager/reservations`
- **Receptionist**: Go to `/receptionist/reservations`

### Step 2: View Your Statistics
You'll immediately see:
- 📊 5 colorful gradient cards showing key metrics
- 📈 Occupancy rate progress bar
- 📋 List of all your bookings
- 📊 Trend charts at the bottom

### Step 3: Create Your First Booking
1. Click the **"+ New Reservation"** button (top right)
2. Fill in guest details
3. Select room (watch the price calculate automatically!)
4. Choose dates
5. Click **"Create Booking"**
6. Done! ✅

---

## 🎯 Most Common Actions

### Check-in a Guest
1. Find the booking in the table
2. Click the **blue check-in button** (🔵)
3. Guest is now checked in!
4. Room automatically marked as OCCUPIED

### Check-out a Guest
1. Find the checked-in booking
2. Click the **cyan check-out button** 
3. Guest is checked out!
4. Room automatically marked as AVAILABLE

### Search for a Booking
Type in the search box:
- Guest name: "John"
- Phone: "17"
- Booking ID: "BK2026"

### Filter Bookings
Use the filter dropdowns:
- **Status**: Confirmed, Checked In, etc.
- **Payment**: Paid or Pending
- **Dates**: Select date range

## 🎨 What You'll See

### Colorful Cards at Top
- **Purple** = Total Bookings
- **Pink** = Today's Check-ins
- **Cyan** = Today's Check-outs
- **Orange** = Pending Payments
- **Teal** = Monthly Revenue

### Color-Coded Badges
- **Green** = Confirmed / Paid
- **Orange** = Pending Payment
- **Blue** = Checked In
- **Gray** = Checked Out
- **Red** = Cancelled

### Action Buttons
- **Blue** = Check-in
- **Cyan** = Check-out
- **Info Blue** = View Details
- **Yellow** = Edit
- **Red** = Delete

## 🆘 Quick Troubleshooting

**No bookings showing?**
- Check you're logged in with the right hotel account
- Try clearing filters

**Can't check-in?**
- Booking must be in "Confirmed" status
- Check booking date is valid

**Can't see certain buttons?**
- Check your role permissions
- Receptionists have limited access

**Charts not showing?**
- Refresh the page
- Check internet connection (Chart.js uses CDN)
